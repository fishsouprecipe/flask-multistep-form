## Pre-requirements
- docker
- docker-compose

## Flow
### Docker commands
- Build images
```
docker-compose build
```
- Gracefully stop and remove all containers
```
docker-compose stop && docker-compose rm
```
- Run the application (the app would be available on http://127.0.0.1:80/)
```
docker-compose up
```
